# 论文结果优化指南

## 📚 推荐资源

### 专业写作 Prompt 库
以下资源提供专业的 Prompt 模板，用于改进论文的表达、逻辑和学术规范：

1. **[Research Writing Skill](https://github.com/alfonso0512/research-writing-skill)**
   - 专注于研究论文的写作技巧
   - 提供系统的优化 Prompt

2. **[Awesome AI Research Writing](https://github.com/Leey21/awesome-ai-research-writing)**
   - AI 辅助学术写作资源集合
   - 包含多种优化工具和方法

## 🎯 优化方向

| 优化类型 | 说明 |
|---------|------|
| **语言润色** | 改进用词、句式和表达的专业性 |
| **逻辑优化** | 增强论证的严谨性和层次感 |
| **格式规范** | 符合学术规范和期刊要求 |
| **内容补充** | 丰富关键部分的论述深度 |

## 💡 使用流程

1. 将生成的论文内容与 Prompt 模板结合
2. 使用 LLM 进行多轮润色迭代
3. 逐步优化文章质量
4. 最终完成学术规范要求的论文

---

**提示**: 不同期刊/会议可能有不同要求，建议参考目标期刊的投稿指南。
