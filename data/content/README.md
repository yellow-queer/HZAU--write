# 项目内容文件夹

此文件夹用于存放项目相关的参考内容文件，系统会自动读取并索引这些文件，辅助 AI 进行论文写作。

## 支持的文件格式

### 文本文件
- `.txt` - 纯文本文件
- `.md` - Markdown 文档

### 文档文件
- `.pdf` - PDF 文档
- `.docx` - Word 文档

### 表格文件
- `.xlsx` - Excel 2007+ 文件
- `.xls` - Excel 97-2003 文件

### 图像文件
- `.jpg` / `.jpeg` - JPEG 图像
- `.png` - PNG 图像

## 使用方法

1. 将项目相关的文件放入此文件夹
2. 运行系统时选择"加载内容库"
3. 系统会自动读取并索引这些文件
4. AI 在写作时会参考这些内容

## 建议放入的内容

- 项目介绍文档
- 实验设计方案
- 数据分析结果
- 实验图表和截图
- 技术文档
- 项目报告
- 会议记录
- 其他参考资料

## 注意事项

1. **图像处理**: 图像文件会使用 Qwen-VL API 进行描述，会产生 API 费用。如不需要，可在运行时禁用图像处理。

2. **文件大小**: 建议单个文件不超过 10MB，过大的文件可能导致处理缓慢。

3. **编码问题**: 文本文件建议使用 UTF-8 编码，系统也支持 GBK 等常见中文编码。

4. **Excel 文件**: 系统会读取每个工作表的前 200 行数据。

5. **文件组织**: 可以在此文件夹下创建子文件夹来组织不同类型的内容，系统会递归读取所有子文件夹中的文件。

## 示例

```
data/content/
├── 项目介绍.txt           # 项目背景和目标
├── 实验设计.docx          # 实验方案文档
├── 实验结果.xlsx          # 实验数据表格
├── 结果分析.pdf           # 分析报告
├── 实验截图/              # 实验结果截图
│   ├── 准确率曲线.png
│   └── 混淆矩阵.jpg
└── 技术文档/              # 技术相关文档
    └── API文档.md
```
